# .jscpd.json

## Purpose

Copy-paste detection config for the landing page sources (`check:dup` in
the `check:all` gate). Mirrors the sibling apps: threshold 0.5 %, minimum
50 tokens, `src/**/*.ts` only, tests and declaration files ignored.

## Invariants & assumptions

- Procedural scene-building code (`src/scene/`) tends toward similar mesh
  factories — shared helpers (`clayMesh`, `namedGroup` in
  `scene/palette.ts`) keep it under the threshold; prefer extending those
  over widening this config.

## Usage

```bash
pnpm run check:dup   # jscpd --config config/.jscpd.json src
```

## Tests

No unit test; exercised by the `check:all` gate inside `pnpm test`.
